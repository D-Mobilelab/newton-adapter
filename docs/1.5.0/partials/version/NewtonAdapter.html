NewtonAdapter

Description

Adapter for Newton sdk to be used in B! web applications

Methods

  • init(options)

    Initializes Newton sdk and sets up internal configuration.

    Parameters
    ParamTypeDetails
    optionsObject

    configuration object

    PropertyTypeDetails
    secretIdstring

    secret id of the application

    enableboolean

    enable or disable calls to Newton library

    (default: false)

    waitLoginboolean

    track events, heartbeats and rankings only after login
    If true, you have to call login() in all cases, both for logged and unlogged users.

    (default: false)

    newtonversioninteger

    version of Newton, it can be 1 or 2.

    (default: 2)

    loggerObject

    logger object containing the methods: debug, log, info, warn, error

    (default: disabled logger)

    propertiesObject

    custom data for Newton session
    Newton version 1: this property is not supported

    (default: {})

    Returns
    PromiseLite

    promise that will be resolved when the init has been completed

    Example

    NewtonAdapter.init({
    secretId: '123456789',
    enable: true,
    waitLogin: true,
    version: 2,
    logger: console,
    properties: {
        hello: 'World'
    }
});

  • isInitialized()

    Check if NewtonAdapter is initialized.

    Returns
    boolean

    true if NewtonAdapter is already initialized (you have called init method)

    Example

    NewtonAdapter.isInitialized();

  • isUserLogged()

    Check if the user is already logged on Newton.
    If called before init, this method returns false.

    Returns
    boolean

    true if the user is already logged on Newton, else false

    Example

    NewtonAdapter.isUserLogged();

  • login(options)

    Performs custom or external login via Newton sdk.
    If you set waitLogin=true on init method, you have to call this method in all cases, for logged and unlogged users.

    Parameters
    ParamTypeDetails
    optionsObject

    configuration object

    PropertyTypeDetails
    typestring

    type of Newton login used, it can be 'custom' or 'external'
    Newton version 1: external login not supported

    (default: "custom")

    loggedboolean

    true if user is logged, false if user is unlogged

    (default: false)

    userPropertiesObject

    custom user properties

    (default: {})

    Returns
    PromiseLite

    promise that will be resolved when the login has been completed

    Example

    NewtonAdapter.login({
      logged: true,
      type: 'external',
      userId: '123456789',
      userProperties: {
          msisdn: '+39123456789',
          type: 'freemium'
      }
});

  • rankContent(options, contentId, scope, score)

    Performs content ranking via Newton sdk
    Newton version 1: feature not supported

    Parameters
    ParamTypeDetails
    optionsObject

    configuration object

    contentIdstring

    unique identifier of the content

    scopestring

    type of action performed on the content

    scorenumber

    the score associated to the content

    Example

    NewtonAdapter.rankContent({
      contentId: '123456777',
      scope: 'social',
      score: 4
});

  • startHeartbeat(options)

    Performs timed events via Newton sdk.

    Parameters
    ParamTypeDetails
    optionsObject

    configuration object

    PropertyTypeDetails
    namestring

    name of the timed event

    propertiesObject

    details of the timed event

    (default: {})

    Example

    NewtonAdapter.startHeartbeat({
      name: 'Playing',
      properties: {
          category: 'Game',
          content: 'Fruit Slicer'
      }
  });

  • stopHeartbeat(options)

    Stops timed events via Newton sdk.

    Parameters
    ParamTypeDetails
    optionsObject

    configuration object

    PropertyTypeDetails
    namestring

    name of the timed event

    propertiesObject

    details of the timed event

    (default: {})

    Example

    NewtonAdapter.stopHeartbeat({
      name: 'Playing',
      properties: {
          category: 'Game',
          content: 'Fruit Slicer'
      }
  });

  • trackEvent(options)

    Performs event tracking via Newton sdk.

    Parameters
    ParamTypeDetails
    optionsObject

    configuration object

    PropertyTypeDetails
    namestring

    name of the event to track

    propertiesobject

    custom datas of the event

    (default: {})

    rankobject

    rank event datas. Newton version 1: feature not supported

    (default: {})

    Example

    NewtonAdapter.trackEvent({
      name: 'Play',
      properties: {
          category: 'Game',
          content: 'Fruit Slicer'
      },
      rank: {
          contentId: '123456777',
          scope: 'social',
          score: 4
      }
});

  • trackPageview(options)

    Performs pageview tracking via Newton sdk.

    Parameters
    ParamTypeDetails
    optionsObject

    configuration object

    PropertyTypeDetails
    propertiesObject

    Properties of the pageview

    properties.urlstring

    url of pageview

    (default: window.location.href)

    Example

    NewtonAdapter.trackPageview({
      properties: {
          url: 'http://www.google.it',
          title: 'Game',
          hello: 'World'
      }
});
