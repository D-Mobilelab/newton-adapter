NewtonAdapter

Description

Adapter for Newton sdk to be used in B! web applications

Methods

  • init(options)

    Initializes Newton sdk and sets up internal configuration

    Parameters
    ParamTypeDetails
    optionsObject

    configuration object

    PropertyTypeDetails
    secretIdstring

    secret id of the application

    enabledboolean

    true if and only if Newton tracking is enabled

    loggerLogger

    any object containing the following methods: debug, log, info, warn, error

    propertiesObject

    custom data for Newton getSharedInstanceWithConfig method

    Example

    NewtonAdapter.init({
    secretId: '123456789',
    enabled: true,      // enable newton
    waitLogin: true,    // wait for login to have been completed (async)
    logger: console
});

  • isUserLogged()

    returns whether the user is already logged on Newton

    Returns
    boolean

    true if and only if the user is already logged on Newton

    Example

    NewtonAdapter.isUserLogged();

  • login(options)

    performs custom or external login via Newton sdk

    Parameters
    ParamTypeDetails
    optionsObject

    configuration object

    PropertyTypeDetails
    typestring

    allowed values: 'custom' or 'external'

    loggedboolean

    true if and only if the user is logged on the product

    userPropertiesObject

    an object containing data about the user

    Returns
    PromiseLite

    promise that will be resolved when the login has been completed

    Example

    NewtonAdapter.login({
      logged: true,       // is user logged?
      type: 'external'        // 'external' or 'custom'
      userId: '123456789',    // mandatory for logged user
      userProperties: {
          msisdn: '+39123456789',
          type: 'freemium'
      }
});

  • startHeartbeat(options)

    performs timed events via Newton sdk

    Parameters
    ParamTypeDetails
    optionsObject

    configuration object

    PropertyTypeDetails
    namestring

    name of the timed event

    propertiesObject

    details of the timed event

    Example

    NewtonAdapter.startHeartbeat({
      name: 'Playing',
      properties: {
          category: 'Game',
          content: 'Fruit Slicer'
      }
  });

  • stopHeartbeat(options)

    stops timed events via Newton sdk

    Parameters
    ParamTypeDetails
    optionsObject

    configuration object

    PropertyTypeDetails
    namestring

    name of the timed event

    propertiesObject

    details of the timed event

    Example

    NewtonAdapter.stopHeartbeat({
      name: 'Playing',
      properties: {
          category: 'Game',
          content: 'Fruit Slicer'
      }
  });

  • trackEvent(options)

    performs event tracking via Newton sdk

    Parameters
    ParamTypeDetails
    optionsObject

    configuration object

    Example

    NewtonAdapter.trackEvent({
      name: 'Play',
      properties: {
          category: 'Game',
          content: 'Fruit Slicer'
      },
      rank: {
          contentId: '123456777',
          scope: 'social',
          score: 4
      }
});

  • trackPageview(options)

    performs pageview tracking via Newton sdk

    Parameters
    ParamTypeDetails
    optionsObject

    configuration object

    PropertyTypeDetails
    urlstring

    url of the page we want to track

    (default: window.location.href)

    Example

    NewtonAdapter.trackPageview({
      title: 'Fruit Page',
      url: 'http://www.google.it'
});
